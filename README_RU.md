# Генератор расписания

Это программа на Python для обработки расписания института, которая считывает информацию о занятиях из Excel-файла и экспортирует её в формате PDF после фильтрации по преподавателям.

**[中文版本](README.md)**

## Функциональные возможности

- Чтение данных расписания из Excel-файла
- Автоматический парсинг информации о преподавателях (всего 59 преподавателей, 549 записей о занятиях)
- Фильтрация занятий по преподавателям
- Генерация PDF-расписания с группировкой по дням недели
- Поддержка поиска преподавателей
- Удобный интерфейс командной строки

## Использование

### Способ 1: Использование интерактивного интерфейса (рекомендуется)

```bash
python main.py
```

Программа покажет список преподавателей, вы можете:
- Ввести номер для выбора преподавателя
- Использовать n/N для следующей страницы, p/P для предыдущей страницы при просмотре списка
- Использовать s/S для поиска преподавателя (поддерживается частичное совпадение)
- Использовать q/Q для выхода из программы

### Способ 2: Использование в коде

```python
from schedule_processor import ScheduleProcessor

# Создание процессора
processor = ScheduleProcessor('ITsTiM_Raspisanie_2_polugodie_25-26_bak__pechat.xlsx')

# Загрузка и парсинг данных
processor.load_data()
processor.parse_schedule()

# Получение всех преподавателей
teachers = processor.get_teachers()

# Получение занятий конкретного преподавателя
teacher_name = "Бурцев В."
schedule = processor.get_teacher_schedule(teacher_name)

# Экспорт в PDF
processor.export_to_pdf(teacher_name, "output.pdf")
```

## Описание файлов

- `schedule_processor.py` - Основной класс обработки, содержащий функции загрузки данных, парсинга и генерации PDF
- `main.py` - Интерактивная основная программа
- `requirements.txt` - Список зависимостей Python
- `ITsTiM_Raspisanie_2_polugodie_25-26_bak__pechat.xlsx` - Исходный файл расписания

## Формат вывода

Сгенерированный PDF-файл содержит:
- Имя преподавателя в качестве заголовка
- Занятия, сгруппированные по дням недели (Понедельник, Вторник, Среда, Четверг, Пятница, Суббота)
- Для каждого занятия: время, название предмета, тип занятия (лк/пз/лб), аудитория

## Пример

После выбора преподавателя "Бурцев В." программа создаст файл `schedule_Бурцев_В_.pdf`, содержащий все занятия этого преподавателя.

## Примечания

- Убедитесь, что Excel-файл находится в той же директории
- Формат имени преподавателя: "Фамилия И." (например: Бурцев В.)
- PDF-файлы сохраняются в текущей директории
