# Генератор расписания занятий

Программа для извлечения информации о занятиях из файла Excel и генерации PDF-файлов с индивидуальным расписанием преподавателей.

## Быстрый старт

```bash
python main.py
```

## Требования

- Python 3.8+
- pandas
- openpyxl
- reportlab

## Установка зависимостей

```bash
pip install pandas openpyxl reportlab
```

## Функции

- Интерактивный интерфейс командной строки
- Постраничный просмотр списка преподавателей
- Поиск преподавателей по имени
- Автоматическая генерация PDF-файлов с расписанием
- Экспорт всех расписаний в один PDF-файл

## Использование

1. Запустите программу: `python main.py`
2. Используйте команды:
   - **Номер** — выбрать преподавателя
   - **n/N** — следующая страница
   - **p/P** — предыдущая страница
   - **s/S** — поиск преподавателя
   - **a/A** — экспорт всех расписаний в PDF
   - **q/Q** — выход

## Формат Excel-файла

- Строки 1-2: Заголовки
- Строка 3: Названия групп
- Строка 4: Заголовки таблицы
- Строка 5+: Данные о занятиях
- Столбец 1: День недели
- Столбец 2: Время
- Столбец 3+: Информация о занятиях

## Формат ячейки занятия

```
Название предмета (тип)
Фамилия И.О. аудитория
```
